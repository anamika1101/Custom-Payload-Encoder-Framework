# Custom Payload Encoder & Obfuscation Framework

## Overview
This project is an educational cybersecurity framework designed to demonstrate how
payload encoding and obfuscation techniques can evade basic signature-based detection
mechanisms.

The project is developed strictly for academic and ethical learning purposes.

---

## Features
- Base64 Encoding
- XOR Encoding
- ROT13 Encoding
- String Obfuscation Techniques
- Signature-Based Detection Simulation

---

## Technologies Used
- Python 3
- Standard Python Libraries

---

## How to Run
1. Download or clone the repository
2. Run the Python file:
   python payload_framework.py
3. Enter a payload such as:
   cmd.exe /c whoami
